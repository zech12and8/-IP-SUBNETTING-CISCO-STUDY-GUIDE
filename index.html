🧮 SUBNETTING FORMULAS
🔢 Number of Subnets: 2^s (where s = bits borrowed from host portion)
🔢 Number of Hosts per Subnet: 2^h - 2 (where h = remaining host bits)
🔢 Block Size / Increment: 256 - subnet mask octet value

📝 Example: /26 = 255.255.255.192 → Block size = 256 - 192 = 64
Subnets: 0, 64, 128, 192 (4 subnets total = 2^2)
Hosts per subnet: 64 - 2 = 62 hosts

🗺️ OSPF ROUTING PROTOCOL (Basics)
OSPF (Open Shortest Path First) — Link-state routing protocol, uses Dijkstra's algorithm

Basic OSPF Configuration:
router ospf 1
network 192.168.1.0 0.0.0.255 area 0
network 10.0.0.0 0.255.255.255 area 0

OSPF Areas:
• Area 0 = Backbone area (must exist)
• Area 1,2,3... = Standard areas connected to Area 0
• Wildcard mask = 0.0.0.255 (matches last octet) = inverse of subnet mask
